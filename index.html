📚 About This Project

GuardianLink is an IoT-based real-time geofencing system developed as an academic mini-project. An ESP32 microcontroller paired with a GPS Neo-6M module continuously tracks the patient's location and streams it to Google Firebase. This web dashboard subscribes to those updates and renders the live position on a Leaflet.js map. If the patient exits the configured geofence radius, an instant alert is shown — enabling caregivers to respond quickly.
