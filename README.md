# GuardianLink - Real-Time GPS Tracker Dashboard
A responsive web-based monitoring interface designed for IoT tracking systems.

### Features
* **Real-Time Visualization:** Integrated with Leaflet.js for live coordinate mapping.
* **Responsive Design:** Built using Tailwind CSS for cross-device compatibility.
* **Hardware Integration:** Designed to interface with ESP32 and GPS/GSM modules via Firebase.
* **Battery & Status Monitoring:** Custom icons and UI components for real-time device health.
